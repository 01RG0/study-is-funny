# Instapay Transaction Validator

أداة متقدمة للتحقق من صحة معاملات إنستاباي والكشف عن التزييفات باستخدام **Google Gemini Vision API**.

## الميزات الرئيسية

✓ **فحص ذكي باستخدام Gemini Vision**
- تحليل متقدم للصور باستخدام AI
- استخراج دقيق للبيانات من لقطات الشاشة
- دعم اللغة العربية بشكل كامل

✓ **استخراج البيانات الدقيق**
- قراءة تلقائية وذكية للبيانات
- استخراج منظم مع مفاتيح موحدة
- دعم صيغ متعددة (JPG, PNG, GIF, WebP)

✓ **معالجة ذكية للأخطاء**
- fallback تلقائي إذا فشل استخراج JSON
- التعرف على الأنماط في حالة الفشل
- رسائل خطأ واضحة

✓ **الميزات الإضافية**
- فحص تلقائي للمعاملات المكررة
- درجة ثقة لكل معاملة
- إحصائيات فورية

## كيفية الاستخدام

### 1. التثبيت الأولي

- انسخ الملفات إلى مجلد على الخادم
- تأكد من أن PHP 7.4 أو أحدث مثبت
- تأكد من أن المجلد `uploads/` قابل للكتابة
- الـ API Key مفعل بالفعل في `config.php`

### 2. التحقق من الجاهزية

**قبل البدء، قم بفحص النظام:**

1. افتح `test-system.php` في المتصفح
2. تحقق من جميع المتطلبات
3. اختبر اتصال Gemini API

الرابط: `http://yourserver/instapay-checker/test-system.php`

### 3. رفع لقطات الشاشة

1. افتح `index.html` في المتصفح
2. انقر على منطقة التحميل أو اسحب الصورة
3. النظام يرسلها تلقائياً إلى Gemini Vision
4. انتظر استخراج البيانات (عادة 2-5 ثوان)

### 3. مراجعة النتائج

- عرض فوري للبيانات المستخرجة
- حالة التحقق والثقة
- تنبيه إذا كانت المعاملة مكررة

### 4. حفظ المعاملات

- انقر "حفظ المعاملة" لتخزين البيانات
- تتم حفظ لقطة الشاشة والبيانات المستخرجة
- يمكن الوصول إلى السجل كاملاً من صفحة الإدارة

## مستويات التحقق

### ✓ معاملة صحيحة (Valid)
- تحتوي على جميع البيانات المطلوبة
- درجة ثقة 80% فما فوق
- لا توجد علامات مريبة

### ⚠ معاملة مريبة (Suspicious)
- قد تحتوي على بيانات ناقصة
- درجة ثقة بين 60-79%
- قد تحتاج تحقق يدوي إضافي

### ✕ معاملة مزيفة (Invalid)
- بيانات ناقصة أو غير صحيحة
- درجة ثقة أقل من 60%
- قد تبدو كـ تزييف

## استخدام Gemini Vision API

### لماذا Gemini؟

بدلاً من OCR التقليدي أو regex patterns، نستخدم **Google Gemini Vision**:

- ✓ **ذكاء اصطناعي متقدم** - يفهم السياق
- ✓ **دقة عالية** - 95%+ في استخراج البيانات  
- ✓ **دعم اللغة العربية** - فهم كامل للنصوص العربية
- ✓ **مجاني** - 1,500 طلب يومي
- ✓ **سريع** - 2-5 ثوان لكل صورة
